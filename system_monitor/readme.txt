================
system_monitor


Application Version
===================
0.1


NCOS Devices Supported
======================
ALL


External Requirements
=====================
None


Application Purpose
===================
Get various system diagnostics, alert on thresholds, and put current status in asset_id field

Monitors CPU, memory, uptime, ethernet port status (including LAN) and alerts at 80% CPU, 80% memory, and on ethernet port status change.

Blocks of code show examples for monitoring parameters so you can write your own.
